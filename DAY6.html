HTML MEDIA & HOSTING With HTML

First Principle: The Web is More Than Text and Images

The web should deliver any kind of content, not just text and pictures. In the past, video and audio required plugins like Flash, QuickTime, or Silverlight, which were inefficient and insecure.

The Core Problem

How can we embed media in a standardized, native, and plugin-free way that works on every browser and device?

The Solution: HTML5 <video> and <audio> Tags

HTML introduced dedicated tags for media: <video> and <audio>.

Part 1: The Basics - Getting a Video on the Page

<!-- Basic video -->
<video src="my-awesome-video.mp4"></video>

Problem: This only shows the first frame, no controls.

Solution: Add the controls attribute.

<video src="my-awesome-video.mp4" controls></video>

Result: A fully functional player with play, pause, timeline, and volume.

Part 2: Intermediate - More Control with Attributes

  • width & height: Set dimensions.
<video src="video.mp4" width="640" height="360" controls></video>
  • autoplay (requires muted):
<video src="video.mp4" autoplay muted controls></video>
  • loop: Restarts video automatically.
  • poster: Thumbnail before play.
<video src="video.mp4" poster="images/video-thumbnail.jpg" controls></video>

Part 3: The <audio> Tag

Works like video but audio-only.

<p>Listen to our theme song:</p>
<audio src="audio/theme-song.mp3" controls></audio>

Part 4: Advanced & Professional

Problem 1: Different Browsers, Different Formats

Solution: Use <source> elements.

<video controls poster="images/thumbnail.jpg">
  <source src="videos/my-video.webm" type="video/webm">
  <source src="videos/my-video.mp4" type="video/mp4">
  Sorry, your browser doesn't support videos.
</video>

Problem 2: Accessibility

Solution: Use <track> for captions and subtitles.

<video controls>
  <source src="video.webm" type="video/webm">
  <source src="video.mp4" type="video/mp4">
  <track src="captions_en.vtt" kind="captions" srclang="en" label="English">
  <track src="subtitles_es.vtt" kind="subtitles" srclang="es" label="Espa√±ol">
